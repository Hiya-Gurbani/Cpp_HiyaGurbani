Commands:

To Compile the plugin.cpp:
g++ -fPIC -c Plugin.cpp -o Plugin.o

To Create the library:
g++ -shared -o libplugin.so Plugin.o

To Link the program:
g++ main.cpp -ldl -o main.exe

To Run the Program:
./main.exe

Note: If we do ldd, it won't show this because Explicit dynamic linking does not 
create a direct dependency on the shared library.
ldd lists shared libraries linked implicitly to the executable.
